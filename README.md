## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/<your-username>/kafka-springboot-tutorial.git
```

### 2. Navigate to the Project

```bash
cd kafka-springboot-tutorial
```

### 3. Start Kafka

```bash
docker compose up -d
```

### 4. Verify Running Containers

```bash
docker ps
```

### 5. Stop Kafka

```bash
docker compose down
```
